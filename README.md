b) Explaining the strucuture of the problem statement solved -

I) Major Modules - 

  1) Movie Catalog Management
  2) Theatre Management
  3) Seating Management
  4) Booking Management
  5) Payment Gateway Integration

II) Architecture - 
The application uses Microservices architercture to make it robust, scalable, and easy to maintain appliaction. Each microservice is responsible for individual task and different microservices communicate with each other using REST APIs. The following microservices to be implemented are:

  1) Movies Microservice
  2) Theatre Microservice
  3) Seating Microservice
  4) Booking Microservice
  5) Payment Microservice
  6) Gateway Microservice
